# Notas para revisão da Apple

Code Quest é um aplicativo educacional para aprender programação por missões, aulas e desafios.

## Como navegar

1. Entre com uma conta Supabase real ou use a conta de demonstração fornecida manualmente no App Store Connect.
2. No Hub, toque em Continuar Jornada para seguir a recomendação calculada.
3. Acesse Campanha para ver capítulos e missões.
4. Acesse Academia Dev para aulas estruturadas.
5. Acesse Arena de Código e Code Lab para prática.
6. Acesse Professor Byte para ajuda didática. As respostas podem conter erros e devem ser conferidas.
7. Acesse Configurações > Enviar feedback para salvar feedback local.
8. Acesse Perfil > Segurança > Excluir minha conta para iniciar o fluxo de exclusão, quando a Edge Function estiver implantada.

## Compras

Nenhuma compra real, assinatura ou pagamento externo está disponível nesta versão. Recursos Premium incompletos foram ocultados em produção.

## Offline

Parte do app funciona offline com progresso local e fallback do Professor Byte. Login, sync e IA real dependem de rede.

## Bloqueio antes da submissão

Preencher credenciais da conta de demonstração no App Store Connect. Não salvar senha no repositório.
