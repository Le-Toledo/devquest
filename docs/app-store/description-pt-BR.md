# Code Quest

Code Quest é um app educacional gamificado para aprender programação em português.

O app combina campanha, aulas, desafios práticos, revisão de erros, progresso por XP e moedas conquistadas dentro do jogo. O Professor Byte ajuda a explicar conceitos e erros, com fallback offline quando a IA não está disponível.

## Recursos principais

- Campanha com capítulos e missões.
- Academia Dev com trilhas de programação.
- Arena de Código com desafios práticos.
- Code Lab com rascunhos e validação local.
- Laboratório de Revisão para praticar erros.
- Ranking, conquistas, moedas internas e progresso sincronizado.
- Central de Feedback local para registrar problemas durante o beta.

## Importante

Nesta versão, não há compra real, assinatura ativa ou pagamento externo. Recursos comerciais incompletos permanecem ocultos no build de produção.
