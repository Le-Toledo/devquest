Dados tratados

O Code Quest usa Supabase Auth para autenticação, podendo tratar e-mail e identificador de usuário. O app também salva progresso, XP, moedas, aulas, desafios, ranking, streak, conquistas, preferências e dados locais de uso educacional.

Professor Byte e IA

Quando o usuário envia uma pergunta ao Professor Byte, o conteúdo pode ser enviado para uma Edge Function Supabase e processado por um provedor externo de IA. O usuário não deve enviar senhas, tokens, chaves ou dados pessoais sensíveis.

Finalidades

Usamos os dados para login, sincronização de progresso, ranking, continuidade da jornada de aprendizado, suporte, segurança e melhoria da experiência educacional.

Terceiros

O app utiliza Supabase para autenticação, banco de dados e Edge Functions. O Professor Byte pode usar OpenRouter por meio de função de servidor. Chaves privadas não ficam no aplicativo móvel.

Exclusão

O usuário pode solicitar a exclusão da conta dentro do app em Perfil > Segurança. A exclusão real depende da função remota estar implantada e validada no ambiente de produção.
