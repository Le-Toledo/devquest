# Release Candidate Checklist

## Bloqueadores

- [ ] Edge Function `delete-account` implantada e testada.
- [ ] Política de privacidade publicada em URL pública.
- [ ] Página pública de suporte publicada.
- [ ] Conta de demonstração criada e informada no App Store Connect.
- [ ] Teste físico em iPhone real.
- [ ] Build/Archive validado no Mac.
- [ ] Screenshots reais capturadas.
- [ ] App Privacy preenchido no App Store Connect.
- [ ] Número de build confirmado maior que o último enviado.

## Localmente validado

- [ ] Testes automatizados.
- [ ] Lint.
- [ ] Typecheck.
- [ ] Audits de conteúdo.
- [ ] Premium/IAP incompleto oculto em produção.
- [ ] Feedback local sem promessa de envio remoto.
- [ ] Professor Byte com aviso de IA.
